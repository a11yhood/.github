
# Security Policy

We are a small community still learning. If you encounter a problem please tell us! 


## Supported Versions
Only the `main` branch is actively supported.

## Reporting a Vulnerability

Please **do not open a public issue**.

Email: create-contact@uw.edu  
or use GitHub’s “Report a vulnerability” feature.


Please include as much as you are able of the following:
- A description of the issue
- Steps to reproduce
- Potential impact


## Disclosure
We will acknowledge receipt and work with the reporter
to disclose responsibly once a fix is available.
