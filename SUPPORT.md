# Support

## Getting Help
- Bugs: open a GitHub Issue
- Questions: use [GitHub Discussions](https://github.com/a11yhood/community/discussions) in the [community](https://github.com/a11yhood/community) repository.

## Scope of Support
We are working to develop community support for using A11yhood, and hope to release datasets in the future. Right now we can provide installation and basic usage. 
We are a small team with limited resources and may not always respond quickly.
